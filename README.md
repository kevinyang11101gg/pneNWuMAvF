# 项目简介

## 核心功能
- 数据清洗：格式统一、缺失值处理、异常值处理、数据类型转换与去重
- 数据预处理：出版、价格、销售表现、主题特征预处理
- 数据整合
- 数据分析
- 数据可视化

## 适用场景
- 图书销售分析
- 热门图书推荐
- 数据可视化研究

## 技术特点
- 使用matplotlib、pandas、numpy等Python库进行数据处理和可视化
- 数据量：10000条
- 代码可运行，根据步骤配置环境即可执行
